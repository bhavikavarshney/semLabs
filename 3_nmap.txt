IP Address:
> ipconfig

Scan IP Address:
> nmap <ipaddress>

Scan host
>> nmap www.google.com

Scan range of IP addresses:
>> nmap -p 1-50 <ipaddress>

Scan port which is open:
>> nmap -p 445 <ipaddress>

Ping scan:
>> nmap -sP <ipaddress>

Scan top ports/ find fixed number of port:
>> nmap -top-ports 10 <ipaddress>

Scan open ports:
>> nmap -open <ipaddress>

Scan common ports:
>> namp -F <ipaddress>

Perform sync scan:
>> nmap -sS <ipaddress>

Scan UDP ports:
>> nmap -sU <ipaddress>

Service version detection:
>> nmap -sV <ipaddress>

Scan script + version detection:
>> nmap -sV -sC <ipaddress>
